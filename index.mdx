# Welcome to Diffusion Studio Pro

When you open Diffusion Studio Pro for the first time, make sure to select a backup folder. All generated assets will be saved there.

<figure><img src="https://1998737141-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FkZmAMkk6kBVPNceQRurL%2Fuploads%2FPx8ElKYqQcx6wsPEF0D1%2Fbackup_folder.gif?alt=media&token=c2042cea-9597-41af-be72-e49a53944c01" alt=""/><figcaption></figcaption></figure>

> Be aware: assets you created in previous projects will no longer be available if you change the backup folder later.

Let’s jump into the editing UI by creating a new project.

<figure><img src="https://1998737141-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FkZmAMkk6kBVPNceQRurL%2Fuploads%2FJZzfzQFEm41KpRFicXdZ%2FTimeline.png?alt=media&token=4c692979-cbc9-4280-ba70-697d269a282b" alt=""/><figcaption></figcaption></figure>

**The 5 panels serve the following functions:**

- **Upper left**: Common video editing tools such as transitions, captions, shapes, and more
- **Lower left**: The media panel, where all assets for your video are stored
- **Top center**: The canvas, showing the current state of your composition; it can be panned and repositioned
- **Bottom center**: A non-linear timeline that supports an infinite number of layers and clips
- **Right:** A chat sidebar for generating assets or manipulating the composition

<figure><img src="https://1998737141-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FkZmAMkk6kBVPNceQRurL%2Fuploads%2FM2MnZqwXxWkLhn227XsB%2Fcollapse_sidebar.gif?alt=media&token=92e01741-2a45-48f1-bab1-9e48636fd7a0" alt=""/><figcaption></figcaption></figure>

The chat sidebar can be hidden anytime for a better timeline experience.

In the status bar, you’ll also find the option to switch to the canvas view.

<figure><img src="https://1998737141-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FkZmAMkk6kBVPNceQRurL%2Fuploads%2F8jIxhNtJx1h3Ar3r4X56%2Fcanvas_tab.gif?alt=media&token=c7ea241d-3318-43c7-be19-62a936ea984b" alt=""/><figcaption></figcaption></figure>

**When to use which page?**

- **Canvas**: Best for generating single assets (images, videos, music) in parallel. Chats are no blocking, i.e. you can submit prompts without waiting for the result of the previous prompt. The canvas also lets you organize assets in whatever structure fits your project.
- **Timeline**: Best for composing assets into a full video. The chat can access the current state of the composition and supports multi-step instructions.

> Rule of thumb: use the timeline for agentic workflows and the canvas for single-asset generation.
